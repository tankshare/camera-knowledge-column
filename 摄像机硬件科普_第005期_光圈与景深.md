# 摄像机硬件科普系列 · 第005期

# 光圈与景深——F值的意义、大光圈vs小光圈的应用场景

> **系列导读**：前四期我们依次讲解了传感器、像素与分辨率、光学变焦、帧率与快门。本期进入成像基础的第五讲——光圈与景深。光圈是摄像机镜头的"瞳孔"，决定了进光量、景深和画质风格，是摄像机选型中最常被忽视却至关重要的一个参数。

---

## 第一层：入门理解——光圈就是镜头的"瞳孔"

想象一下，你的眼睛在黑暗环境里瞳孔会放大，在强光下瞳孔会缩小——这正是光圈在摄像机里做的事情。

**光圈（Aperture）**，就是镜头内部控制进光量的可调节开孔。孔越大，单位时间内进入传感器的光线越多，画面越亮；孔越小，进光越少，画面越暗。

光圈用 **F值（F-number / f-stop）** 来表示，比如 F1.8、F2.8、F4.0、F8.0。这里有一个反直觉的地方：

> **F值越小 → 光圈越大 → 进光量越多**
> **F值越大 → 光圈越小 → 进光量越少**

就像数学里的分数：F1.8 = 1/1.8，F8 = 1/8，分母越大，孔径越小。

另一个由光圈决定的视觉效果是**景深**——简单说就是画面里有多少范围是清晰的。大光圈（低F值）会产生**浅景深**，主体清晰、背景虚化（就是手机拍人像时的"背景模糊"效果）；小光圈（高F值）则产生**深景深**，从近到远都很清晰。

## 第二层：进阶参数——F值、T值、最大光圈与光圈叶片数

### F值与T值的区别

| 参数 | 说明 | 应用场景 |
|------|------|----------|
| **F值** | 理论光圈比值（焦距/孔径直径），不考虑镜片透光损耗 | 消费级/商业摄像机标注 |
| **T值（Transmission）** | 实际透光量，考虑镜片玻璃的折射吸收损耗 | 电影镜头、广播级镜头 |

> 对于普通会议摄像机（如 MG200系列、UV417），标注的都是F值。广播级设备才会标注T值。通常T值比F值高0.3~0.5档（即实际进光量略少于理论值）。

### 常见摄像机光圈参数一览

| 产品类型 | 典型光圈范围 | 说明 |
|----------|------------|------|
| 消费级Webcam（MG104等） | F2.0～F2.8（固定） | 固定光圈，无光圈调节功能 |
| 会议摄像机（MG200系列） | F1.8～F2.0 | 大光圈设计，低照度优先 |
| PTZ摄像机（UV417） | F1.8～F4.0（变焦联动） | 变焦时光圈随焦距变化 |
| 广播级摄像机 | F1.4～F11（可调） | 支持光圈环手动调节 |
| 工业检测摄像机 | F2.8～F16（可调） | 需要极大景深，常用小光圈 |

### 光圈叶片数的影响

镜头内部的光圈由多片金属叶片（虹膜叶片）组成：
- **叶片越多**（7片、9片、11片）：光圈开口越接近圆形，背景虚化的焦外光斑（Bokeh）越圆润自然
- **叶片越少**（5片、6片）：光圈趋近多边形，焦外光斑呈多边形，画面有些"廉价感"

会议摄像机因为不追求虚化效果，叶片数通常不是设计重点；但在直播和影视制作领域，圆形焦外是高端镜头的重要标志。

### 光圈与曝光三角形

光圈与上一期讲过的**快门速度**，以及**ISO感光度**，共同构成摄影/摄像中著名的**曝光三角形**：

```
曝光量 = 光圈 × 快门速度 × ISO
```

- 想在暗光下拍清楚，可以开大光圈（F1.8）、降低快门速度（1/30s）、提高ISO（但会增加噪点）
- 三者相互制约，摄像机的自动曝光（AE）系统会根据环境实时平衡这三个参数

## 第三层：专业原理——镜头成像、景深公式与摄像机光圈设计

### 景深（Depth of Field，DOF）的决定因素

景深不只由光圈决定，它是以下四个因素共同作用的结果：

$$\text{景深} \propto \frac{弥散圆 \times 物距^2}{焦距^2 \times 光圈值}$$

具体影响规律：

| 变量 | 变化方向 | 对景深的影响 |
|------|---------|------------|
| 光圈F值 | 增大（缩小光圈） | 景深增大（更多范围清晰） |
| 焦距 | 增大（长焦端） | 景深减小（背景更虚） |
| 拍摄距离 | 增大（被摄体更远） | 景深增大 |
| 传感器尺寸 | 增大（更大底） | 景深减小（更强虚化） |

**这就解释了为什么手机不容易拍出虚化效果**：手机传感器面积极小（1/2.8英寸），焦距极短（折算后约26mm），即使光圈做到F1.7，景深依然很深，难以实现自然虚化——所以手机需要靠AI计算模拟虚化（Portrait Mode）。

### 变焦镜头的光圈变化：恒定光圈 vs 浮动光圈

PTZ摄像机（如UV417的10X光学变焦镜头）在变焦时，光圈会发生变化：

- **浮动光圈（Variable Aperture）**：广角端F1.8，长焦端自动缩小到F3.5或更小。这是因为变焦镜头在长焦端镜片排列变化，等效通光量降低。大多数消费级和商业级PTZ摄像机采用此方案，成本低。
- **恒定光圈（Constant Aperture）**：无论在哪个焦距，光圈始终保持一致（如F2.8恒定）。需要更复杂的光学设计，成本高，主要见于专业广播镜头和高端影视镜头。

### 摄像机镜头的光学结构与最小弥散圆

镜头设计的核心目标是让从场景中每个点发出的光线，经过镜头折射后精确汇聚在传感器上的一个点。当焦点不在传感器平面上时，就会形成**弥散圆（Circle of Confusion，CoC）**——看起来是模糊的光斑而非清晰的点。

弥散圆直径决定了"可接受的清晰范围"（即景深边界）。传感器越大，允许的弥散圆直径也越大（因为最终显示时放大倍率较低），因此大底摄像机相较同样F值下景深更浅。

### ISP中的自动光圈控制（AAC）

高端摄像机内置**自动光圈控制（Auto Aperture Control）**或通过ISP的AE算法间接控制光圈：
1. ISP持续读取当前帧的亮度直方图（Luminance Histogram）
2. 与目标曝光值（Target Exposure Value）对比
3. 根据偏差量，通过驱动电路调整光圈叶片开合角度
4. 同步调整快门速度和ISO增益，完成三角形平衡

会议摄像机通常优先保持**快门速度在安全范围内**（避免运动模糊），再调光圈，最后才提ISO（控制噪点）。

## 第四层：应用实践——会议场景下如何选择光圈参数

### 场景对比：需要大光圈还是小光圈？

| 应用场景 | 推荐光圈 | 理由 |
|----------|---------|------|
| 小型会议室（低照度，桌面灯光） | **F1.8～F2.0大光圈** | 采光不足，需要更多进光量保持画面亮度 |
| 大型会议室（充足灯光） | F2.0～F2.8 | 光线充足，适当缩小光圈可提升画面边缘锐度 |
| 户外活动直播 | F2.8～F4.0 | 户外光线充足，需要较小光圈避免过曝 |
| 工厂/仓库监控 | F1.4（定焦） | 工业现场光线复杂，大光圈应对暗区 |
| 录播教室（AI追踪） | F2.0（推荐固定大光圈） | 需要兼顾追踪对象清晰度，浅景深可能导致边缘追踪目标模糊 |
| 广电演播室 | 可调F2.8～F8 | 专业照明系统，通过光圈精确控制曝光 |

### 会议摄像机选型：光圈是关键参数之一

以明日（Minrray）产品线为例：

| 产品型号 | 传感器尺寸 | 光圈 | 适用场景 |
|----------|-----------|------|---------|
| MG104（1080P Webcam） | 1/2.9英寸 | F2.0（固定） | 小型会议室、个人办公 |
| MG200C/MG200P（4K Webcam） | 1/2.5英寸 | F1.8（固定） | 中小型会议室、高清直播 |
| MG201（4K USB3.0 Webcam） | 1/2.5英寸 | F1.8（固定） | 需要USB3.0高带宽场景 |
| UV417（4K PTZ双镜头） | 1/2.5英寸 | F1.8～F4.0（变焦浮动） | 大型会议室、追踪广角监控 |

> **选型建议**：会议场景中，光线通常不如户外充足，优先选择**F1.8~F2.0的大光圈**产品。如果会议室配有专业灯光系统（泛光灯、顶置筒灯），F2.0~F2.8已足够，无需过度追求大光圈。

### 光圈与逆光补偿的关系

会议室常见"逆光"问题（参会者背对窗户），此时：
- 即使光圈已经是F1.8，背景（窗外）依然远比人脸亮
- 单靠扩大光圈无法解决，需要配合**背光补偿（BLC）**或**宽动态（WDR）**技术
- WDR原理（下期第006期快门/曝光专题将详细展开）：通过多次曝光合成，同时保留高亮区和暗区的细节

## 第五层：连接与传输——光圈参数如何通过接口协议控制

### 通过控制协议远程调节光圈

在实际部署中，工程师和用户无需手动触碰摄像机镜头，可以通过软件/协议远程控制光圈：

#### VISCA协议（PTZ摄像机主流控制协议）
```
光圈直接命令（VISCA over IP/Serial）：
  开大光圈（Iris Up）：  81 01 04 0B 02 FF
  缩小光圈（Iris Down）：81 01 04 0B 03 FF
  自动光圈（Iris Auto）：81 01 04 0B 00 FF
  手动光圈（Iris Manual）：81 01 04 0B 01 FF
  直接设置光圈值：       81 01 04 4B 0p 0q 0r 0s FF（pqrs为光圈值编码）
```

#### ONVIF协议（网络摄像机标准协议）
通过 `SetImagingSettings` 接口可以设置：
- `Iris.IrisType`：AUTO / FIXED / PIris / DCIris
- `Iris.Value`：具体光圈数值

支持ONVIF的摄像机（如UV417）可通过NVR系统、VMS软件统一管理光圈参数，无需逐台配置。

#### 摄像机Web管理界面
大多数网络摄像机提供HTTP Web UI，支持：
- 曝光模式切换（全自动 / 快门优先 / 光圈优先 / 手动）
- 光圈优先（A档）：固定光圈值，由ISP自动调整快门和ISO
- 全手动（M档）：固定光圈、快门、ISO，适合灯光稳定的演播室场景

### 光圈与视频带宽的间接关系

光圈大小影响画面质量，进而影响编码带宽需求：
- 大光圈（浅景深）→ 背景虚化 → 画面细节减少 → **编码效率更高**（H.265可进一步压缩）
- 小光圈（深景深）→ 全场景清晰 → 画面细节丰富 → **需要更高码率**才能保证画质

在IP摄像机系统中，这意味着部署大景深（小光圈）的摄像机时，需要适当提高码率预算，通常：
- 1080P @ 小光圈全清晰场景：推荐码率 4～8 Mbps（H.264）/ 2～4 Mbps（H.265）
- 4K @ 小光圈全清晰场景：推荐码率 12～20 Mbps（H.264）/ 6～10 Mbps（H.265）

## 本期重点回顾

| 知识点 | 核心要点 |
|--------|---------|
| F值含义 | F值越小→光圈越大→进光量越多；F值越大→光圈越小→景深越深 |
| 景深影响因素 | 光圈、焦距、拍摄距离、传感器尺寸四者共同决定 |
| 恒定vs浮动光圈 | 恒定光圈贵但稳定；浮动光圈（消费/商业级主流）变焦时进光量变化 |
| 会议场景建议 | 低照度优先选F1.8～F2.0大光圈；专业照明环境F2.8已足够 |
| 远程控制 | VISCA协议/ONVIF均支持远程光圈调节，配合Web界面可设光圈优先模式 |
